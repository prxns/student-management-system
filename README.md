# Student Management System

A desktop based Student Management System built using Python, Tkinter, and SQLite.
This application allows users to manage records through a professional graphical user interface.

---

## Features

- Add new student records
- View all students in table format
- Update existing student information
- Delete student records
- Search Students by ID
- Professional table layout using Tkinter Treeview
- Persistent data storage using SQLite database
- Interactive and user-friendly GUI

---

## Technologies Used

- Python
- Tkinter (GUI)
- SQLite (Database)
- Git & Github

---

## How to Run

1. Clone the repository (git clone)
2. Navigate to the project folder (cd)
3. Run the application (gui.py)
